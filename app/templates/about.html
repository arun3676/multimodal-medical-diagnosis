About This Project

Project Overview

The Multimodal AI Medical Diagnosis System is an innovative project that demonstrates the integration of computer vision and large language models for medical image analysis, particularly focused on chest X-rays.

This system was developed as part of an industry-oriented project to explore the potential of AI in supporting medical diagnostics while completing my Master's in Computer Science at Lamar University.

Key Features:

  • Computer Vision Analysis: Utilizes Google Cloud Vision API to extract features and objects from medical images
  • Natural Language Processing: Employs OpenAI's GPT-4 to interpret visual findings and generate preliminary assessments
  • Multimodal Integration: Combines visual data with textual patient information for comprehensive analysis
  • Batch Processing: Supports processing multiple images with progress tracking
  • Report Generation: Creates formatted medical reports for educational purposes

Educational Purpose:

This system is designed as an educational and research tool to demonstrate the capabilities of AI in medical image analysis. It is not intended for clinical use and should not replace professional medical advice or diagnosis.

Implementation Details

Vision Analysis Pipeline:

  1. X-ray image is processed using Google Cloud Vision API
  2. Multiple analysis models extract objects, labels, and features
  3. Results are consolidated into a structured data format
  4. Optional batch processing for multiple images

Medical Interpretation:

  1. Vision results combined with patient symptoms
  2. Data sent to GPT-4 with specialized medical prompts
  3. AI generates preliminary assessments and suggestions
  4. Results formatted into a comprehensive report
Research Impact: This project demonstrates the potential of AI to assist in medical image interpretation while highlighting the importance of human expertise in the final diagnosis.
