# Multimodal AI Medical Diagnosis System

An AI-powered system that analyzes chest X-rays using computer vision and large language models to assist in medical diagnosis.

## Features
- Chest X-ray analysis using Google Cloud Vision API
- Medical interpretation using GPT-4
- Batch processing capabilities
- Professional medical report generation
- Progress tracking and resumable processing

## Prerequisites
- Python 3.10+
- Google Cloud Vision API credentials
- OpenAI API key
